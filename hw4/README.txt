Notes:
1. The files named ".exe" are executable programs in a Unix-like environment. They are not Windows executables.
2. Recommended environment: Linux, macOS, or WSL on Windows.
3. To run the programs:
   cd hw4_1 && ./main.exe
   cd hw4_2 && ./main.exe
   cd hw4_3 && ./main.exe
4. To compile the reports, use XeLaTeX:
   xelatex main.tex
5. Problem 2 uses __float128 and libquadmath. A typical compilation command is:
   gcc main.c -O2 -std=c11 -lquadmath -lm -o main.exe
6. For Problem 2, gnuplot and Python are both used to generate the figures. They are mutually replaceable, but in the report the Python-generated figure is used for a better view.
