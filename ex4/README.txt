This archive contains the files for Exercise 4.

Directory structure:

hw3/
├── README.txt
├── 计算物理ex4.pdf      	-- report/analysis for the exercise
├── ex4.c                	-- C source code for the exercise
├── ex4.exe              	-- executable program for the exercise
├── main.tex              	-- LaTeX source file for the exercise report
├── plot.gp			-- executable file for plotting the figures, output by the c file
├── poly_zoom1.csv
├── poly_zoom2.csv		-- the tables with finely-disposed data for plotting
├── zoom1_double.png
├── zoom1_float.png
├── zoom1_quad.png
├── zoom2_double.png
├── zoom2_float.png
└── zoom2_quad.png		-- figures created by the gp file, meeting the problem

Notes:
1. The files named ".exe" are not Windows executables. The suffix here only indicates that the files are executable programs.
2. Please run these programs in a Unix-like environment, such as Linux, macOS, or WSL on Windows.
3. Run the program with:
   ./ex4.exe
