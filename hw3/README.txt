This archive contains the files for Homework 3.

│   ├── 计算物理hw3_1.pdf      -- report/analysis for Problem 1

│   ├── main.c                -- C source code for Problem 1
│   └── main.exe              -- executable program for Problem 1
│
├── hw3_2/
│   ├── 计算物理hw3_2.pdf      -- report/analysis for Problem 2
│   ├── main.tex              -- LaTeX source file for Problem 2
│   ├── main.c                -- C source code for Problem 2
│   └── main.exe              -- executable program for Problem 2
│
├── hw3_3.4/
│   ├── 计算物理hw3_3_4.pdf    -- report/analysis for Problems 3 and 4
│   ├── main.tex              -- LaTeX source file for Problems 3 and 4
│   ├── 3.c                   -- C source code for Problem 3
│   ├── 3.exe                 -- executable program for Problem 3
│   ├── 4.c                   -- C source code for Problem 4
│   ├── 4.exe                 -- executable program for Problem 4
│   ├── 4f.c                  -- additional/modified C source code for Problem 4
│   └── 4f.exe                -- executable program corresponding to 4f.c
│
└── hw3_5/
    ├── 计算物理hw3_5.pdf      -- report/analysis for Problem 5
    ├── main.tex              -- LaTeX source file for Problem 5
    ├── main.c                -- C source code for Problem 5
    └── main.exe              -- executable program for Problem 5

Notes:
1. The files named ".exe" are not Windows executables. The suffix here only indicates that the files are executable programs.
2. Please run these programs in a Unix-like environment, such as Linux, macOS, or WSL on Windows.
3. For hw3_1, hw3_2, and hw3_5, run the program with:
   ./main.exe
4. For hw3_3.4, run the corresponding program with:
   ./3.exe
   ./4.exe
   or
   ./4f.exe
   depending on which part you want to test.
